# Kingswork

A mobile-friendly, static learning site for exploring kings from the Bible through searchable cards, Scripture references, a timeline, a simplified classroom map, quick checks, and a compare tray.

## What is included

- 50+ profiles covering the united monarchy, the divided kingdoms of Judah and Israel, the fall of Israel, Judah's final kings, exile/return rulers, and major foreign kings who appear in the Bible story.
- Bible references on every profile so students can verify each summary in Scripture.
- Kid-friendly lesson takeaways, status badges, filters, sorting, timeline rows, map filtering, and quick-check prompts.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 4173
```

Then visit <http://127.0.0.1:4173>.

## Check JavaScript

```bash
npm test
```
